# TelegramFaceSwapper
STEP 1.
```
git clone https://github.com/antonVosc/TelegramFaceSwapper.git
```
STEP 2.
```
python3 -m venv venv
```
STEP 3.
```
source venv/bin/activate
```
STEP 4.
```
pip install -r requirements.txt
```
STEP 5. Go to https://t.me/BotFather

STEP 6. Start the bot and follow the instructions to create API key.

STEP 7. Copy HTTP API from BotFather and paste it on line 7 in main.py.

STEP 8.
```
python main.py
```
